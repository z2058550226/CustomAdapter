# CustomAdapter
将LinearLayout或GridLayout变成ListView的一款邪教Adapter
仿照BaseQuickAdapter的写法，只不过把界面更新item、删除item、添加item和item全部刷新的逻辑全部替换成了addView或removeView系列的方法

## 导入方式
`implementation 'com.suika:CustomAdapter:1.0.2'`

## 使用方法
